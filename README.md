# AGRAH Eden

AGRAH Eden is a research and development project focused on building a **semi-autonomous indoor vertical hydroponic farming system** using IoT, automation, and Artificial Intelligence.

The goal is to make home food production easier by enabling anyone to grow fresh vegetables indoors with minimal daily maintenance. The long-term vision includes intelligent monitoring, predictive analytics, mobile control, educational resources, and a community marketplace for sharing or selling harvests.

> **Current Status:** Research & Planning (Prototype MVP)

## Vision

* Indoor vertical hydroponic farming
* IoT-based real-time monitoring
* Semi-autonomous environmental control
* Mobile application for remote monitoring
* AI-assisted plant health analysis
* Community marketplace
* Food security through accessible technology
