# Hardware Roadmap

## MVP Goal

Build a working indoor vertical hydroponic farming prototype using affordable and widely available components.

---

# Core Controller

* ESP32 (Primary Controller)
* Raspberry Pi 5 (Optional AI & Edge Computing)
* Arduino (Optional if additional I/O is required)

---

# Sensors

## Required

* Water Level Sensor
* pH Sensor
* EC/TDS Sensor
* DHT22 or BME280 (Temperature & Humidity)
* Light Sensor (BH1750)
* DS18B20 Waterproof Water Temperature Sensor

## Future

* CO₂ Sensor
* Dissolved Oxygen Sensor
* Water Flow Sensor
* Camera Module (Plant Health Detection)

---

# Actuators

* Water Pump
* Air Pump
* Solenoid Valve
* Relay Module
* LED Grow Lights
* Cooling Fan
* Exhaust Fan

---

# Communication

* Wi-Fi (ESP32)
* Bluetooth (Optional)
* MQTT
* HTTP/REST API

---

# Power

* 12V Power Supply
* 5V Buck Converter
* Battery Backup (Future)

---

# Indoor Hydroponic System

Possible growing methods:

* Vertical NFT (Recommended MVP)
* Vertical DWC
* Tower Hydroponics

---

# Recommended Plants for MVP

* Lettuce
* Pechay
* Kangkong
* Basil
